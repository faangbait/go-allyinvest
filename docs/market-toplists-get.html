GET market/toplists/:list{.format}

Description

This call will return a ranked list based on the list type specified.

Authentication Required
Yes OAuth 1.0a
URI Parameters
list type of list to be requested.
Example URL
https://devapi.invest.ally.com/v1/market/toplists/topvolume.xml
Queryable Fields
exchange
the exchange for the top movers to be requested. The followings are the exchange abbreviation to be assigned: (the default exchange is NYSE)
A: American Stock Exchange
N: New York Stock Exchange
Q: NASDAQ
U: NASDAQ Bulletin Board
V: NASDAQ OTC Other

Special Considerations

Available list types include:

toplosers Top losers by dollar amount
toppctlosers Top percentage losers
topvolume Top volume
topactive Top active
topgainers Top gainers by dollar amount
toppctgainers Top percentage gainers

Data Fields

Data Tag Data Description
chg Change since prior day close (cl)
last Last trade price
name Company name
pchg percentage change from prior day close
pcls Prior day close
rank Rank within list returned
symbol Ticker symbol
vl Today's cumulative volume
