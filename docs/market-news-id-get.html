GET market/news/:id{.format}

Description

This call will return an article identified by the URI id.

Authentication Required
Yes OAuth 1.0a
URI Parameters
id an article id obtained from /market/news/search
Example URL
https://devapi.invest.ally.com/v1/market/news/234899d5fd2ee9a501a8349a0f571f6f.xml

Data Fields

Data Tag Description
id article id used for article retrieval
headline the title of the article
date date and time the article was published
story the body text of the article

Special Considerations

News is only returned for a total of 180 days from the time of request.
