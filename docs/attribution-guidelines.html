Attribution Guidelines

Determining Whether Attribution is Required

If you are leveraging the Ally Invest API for the purposes of displaying data or information, link-back attribution is required. If you are leveraging the Ally Invest API to execute, or allow other users to execute, trades on the Ally Invest platform, we kindly request – but do not require – link-back attribution.

If you have any questions about whether you are providing proper attribution, feel free to send a screenshot to InvestAPI@ally.com and we'll provide you with feedback.

Attribution Guidelines

Once it has been determined that attribution is required, please follow the guidelines below. Please note that any and all attribution links should be “followed” links.

Attribution Placement

The attribution should be placed in the same context as the data/information this is being displayed. In simple terms – please don’t bury the attribution statement in the footer of a page.

Attribution Content

The general attribution statement can be one of 2 variations:

Powered by Ally Invest 

<p>Powered by <a href=http://www.ally.com/invest title=”Powered by Ally”>Ally Invest</a></p>

Provided by Ally Invest 

<p>Provided by <a href=http://www.ally.com/invest title=”Powered by Ally”>Ally Invest</a></p>

In both variations, the term Ally Invest should be clickable and link to http://www.ally.com/invest .

Branding Requirements and Resources

Text Usage

When referencing Ally Invest in a text attribution, please display Ally Invest as two words. For example: Powered by Ally Invest
