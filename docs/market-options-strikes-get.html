GET market/options/strikes{.format}

Description

This call will return the full list of available option strikes for a given symbol.

Authentication Required
Yes OAuth 1.0a
URI Parameters
format a response format, default is xml (See Response Structure )
Query Parameters
symbol a single symbol for which options are traded
Example URL
https://devapi.invest.ally.com/v1/market/options/strikes.xml

Data Fields

Data Tag Description
price A strike price
