GET watchlists/:id{.format}

Description

This will get the watchlist specified in the URI.

Authentication Required
Yes OAuth 1.0a
URI Parameters
format a response format, default is xml (See Response Structure )
id a watchlist ID to retrieve
Example URL
https://devapi.invest.ally.com/v1/watchlists/DEFAULT.xml

GET /v1/watchlists/watchlist-18516.xml HTTP/1.1
